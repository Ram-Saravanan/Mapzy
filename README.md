# Mapzy

AI-powered navigation and road safety platform that combines turn-by-turn navigation, hazard intelligence, and community-driven reporting to improve driver awareness and road safety.

**Live Demo:** https://mapzy-sigma.vercel.app/

## Overview

Mapzy is a navigation application designed to help drivers make safer decisions through real-time hazard visibility and community-sourced road intelligence. The platform integrates routing, hazard reporting, nearby place discovery, and gamification into a single user experience.

The project was developed to address the lack of accessible safety-focused navigation tools that provide both route guidance and contextual road information.

---

## Key Features

### Navigation

* Turn-by-turn navigation using Mappls SDK
* Route planning and live route guidance
* Integrated hazard visualization during navigation

### Hazard Intelligence

* Displays road hazards sourced from OpenStreetMap
* Real-time hazard visibility with map overlays
* Community-reported incidents and road conditions

### Community Reporting

* User-generated hazard reporting
* Moderation and verification workflows
* Contribution tracking and report history

### User Engagement

* Reward points system
* Leaderboards and contribution rankings
* User profiles and activity statistics

### Nearby Services

* Discovery of nearby fuel stations, hospitals, pharmacies, restaurants, banks, ATMs, hotels, and cafes
* Location-based search within configurable radius

### Authentication

* Google Sign-In
* Email and password authentication
* Personalized user profiles and saved search history

* Built a full-stack navigation platform integrating mapping, authentication, and community features.
* Developed a hazard reporting ecosystem powered by user contributions and OpenStreetMap data.
* Implemented gamification mechanisms to encourage high-quality community participation.
* Designed scalable cloud-based architecture using Firebase services.
* Created a unified navigation and safety experience without requiring multiple applications.
